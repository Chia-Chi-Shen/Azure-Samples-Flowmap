Flowmap GL Application This sample shows how to use the flowmap layer on Azure Maps with the flowmap.gl module.
Flowmap layers visualize movement data between two points. The data is displayed as arcs between the points. The thickness of the arcs represents the volume of movement between the points.

The sample uses data from BIXI, a public bike share system in Montreal. The flow map layer visualizes the movement of bikes between stations. Hover over the dots and arcs to see information about the locations and movements.

The code references flowmap.gl's example.
